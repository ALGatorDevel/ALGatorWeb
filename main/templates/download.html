Download ALGator 

ALGator can be used either as a web application or as a standalone desktop application. The primary advantage of the web application is that project data is accessible to all users. However, writing algorithm code within the web interface can be more challenging. When installed as a desktop application, ALGator allows users to edit the project's code using their preferred editor (such as NetBeans, IntelliJ IDEA, etc.). Projects created in the desktop application can later be uploaded to the web, making them available to other users.

To install ALGator as a standalone application, follow the instructions provided for:

  • Windows users
  • Linux/macOS users
