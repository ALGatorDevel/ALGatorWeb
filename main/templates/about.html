About  ALGator 

ALGator automates the algorithm evaluation process by executing algorithm implementations on predefined test case sets and analyzing various performance indicators.

To use ALGator, the user must define a project that includes the problem definition, test case sets, input parameters, output indicators, and evaluation criteria for algorithm quality. Once a project is set up, multiple algorithm implementations can be added. Upon request, the system executes all implemented algorithms, verifies their correctness, and compares the quality of their results.

With ALGator, users can define additional quality criteria, generate graphs, and conduct evaluations and comparisons of the implemented algorithms.

For more information see the how it works section.
